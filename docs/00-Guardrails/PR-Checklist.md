
# PR Checklist

- [ ] Linked Story/Issue
- [ ] AC covered (listed)
- [ ] Tests added/updated (unit/integration/e2e)
- [ ] Security: no secrets, no PII in logs, authz checked if applicable
- [ ] DB migration included (if needed) + rollback steps
- [ ] Observability: logs/metrics updated (no sensitive data)
- [ ] UI evidence (screenshots/video) if UI changes
