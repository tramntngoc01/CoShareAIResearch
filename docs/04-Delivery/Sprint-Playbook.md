
# Sprint Playbook (AI-first safe)

## Story lifecycle
Draft -> Ready -> In Dev -> In QA -> Done

## Story "Ready" checklist
- AC measurable
- Sample data included (fake)
- Edge cases listed
- API mapping known or marked TBD
- Security notes captured (authz/PII/logging)

## Definition of Done
See `docs/00-Guardrails/Definition-of-Done.md`.

## Staging deploy per sprint
- Every sprint produces a staging deploy with smoke test evidence.
